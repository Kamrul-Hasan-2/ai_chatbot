# AI Template Intent/Category Search API

## Overview
The AI Template Intent Search API provides a way to search for product categories and retrieve relevant information from the BDStall database.

## Endpoint

### GET `/api/item/ai_template`

Searches for a product category in the search_intent_items database and returns category information.

## URL Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `intent` | string | Yes | The type of intent. Currently supports: `category` |
| `category` | string | Yes | The product category to search for (e.g., "Laptop", "Desktop PC") |
| `key` | string | Optional | API key for authentication (mkh677ddd2sxxk) |

## Examples

### Request

```bash
# Search for laptop category
curl "http://localhost:8000/api/item/ai_template?intent=category&category=laptop&key=mkh677ddd2sxxk"

# Search for desktop PC
curl "http://localhost:8000/api/item/ai_template?intent=category&category=Desktop%20PC&key=mkh677ddd2sxxk"
```

### Successful Response (200)

When a category is found:

```json
{
  "success": true,
  "data": "আপনি laptop ক্যাটাগরিতে বিভিন্ন পণ্য দেখতে পারেন। এই লিংকে ক্লিক করুন: https://www.bdstall.com/laptop/",
  "category": "Laptop",
  "url": "https://www.bdstall.com/laptop/"
}
```

### Not Found Response (404)

When a category is not found:

```json
{
  "success": false,
  "error": "Category 'your-category' not found in search database",
  "data": null,
  "category": "your-category"
}
```

### Error Response (400)

When required parameters are missing:

```json
{
  "success": false,
  "error": "Missing required parameters: intent and category",
  "data": null
}
```

## Response Format

### Success (Category Found)

```json
{
  "success": true,
  "data": "<Bengali message with category name and link>",
  "category": "<Exact category name>",
  "url": "<BDStall category URL>"
}
```

### Failure (Category Not Found)

```json
{
  "success": false,
  "error": "<Error message>",
  "data": null,
  "category": "<Requested category>"
}
```

## Error Codes

| Status | Description |
|--------|-------------|
| 200 | Category found successfully |
| 400 | Missing or invalid parameters |
| 404 | Category not found in database |
| 500 | Server error (e.g., database file not found) |

## Usage Notes

1. **Case-Insensitive Search**: The search is case-insensitive, so "laptop", "Laptop", and "LAPTOP" will all work
2. **Whitespace Handling**: Leading and trailing whitespace is automatically trimmed
3. **Bengali Response**: The API returns response messages in Bengali for better user experience
4. **URL Generation**: Category URLs are generated by replacing spaces with hyphens in the category name

## Python Example

```python
import requests
import json

# API endpoint
url = "http://localhost:8000/api/item/ai_template"

# Parameters
params = {
    "intent": "category",
    "category": "Laptop",
    "key": "mkh677ddd2sxxk"
}

# Make request
response = requests.get(url, params=params)

# Check response
if response.status_code == 200:
    data = response.json()
    if data["success"]:
        print(f"Category: {data['category']}")
        print(f"Message: {data['data']}")
        print(f"URL: {data['url']}")
    else:
        print(f"Error: {data['error']}")
else:
    print(f"HTTP Error: {response.status_code}")
```

## JavaScript Example

```javascript
// Search for a category
async function searchCategory(category) {
    const url = new URL("http://localhost:8000/api/item/ai_template");
    url.searchParams.append("intent", "category");
    url.searchParams.append("category", category);
    url.searchParams.append("key", "mkh677ddd2sxxk");
    
    try {
        const response = await fetch(url);
        const data = await response.json();
        
        if (data.success) {
            console.log("Category found:", data.category);
            console.log("Message:", data.data);
            console.log("URL:", data.url);
        } else {
            console.log("Error:", data.error);
        }
    } catch (error) {
        console.error("Request failed:", error);
    }
}

// Usage
searchCategory("Laptop");
```

## Integration with Chatbot

This API can be integrated into the main chatbot system to provide category information when users search for products. The API is designed to work seamlessly with the BDStall AI Template system.

## Database File

The API reads from `data/search_intent_items.json`, which contains a comprehensive list of all supported product categories. To add new categories, simply add them to this JSON array.

## Performance

- **Search Time**: O(n) where n is the number of items in the database
- **Typical Response Time**: < 100ms
- **Database Size**: ~1000+ product categories
