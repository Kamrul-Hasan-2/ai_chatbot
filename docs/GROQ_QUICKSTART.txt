INTEGRATION IN CHATBOT
════════════════════════════════════════════════════════════════════════════

The BDStallChatbotSystem automatically detects product searches and routes
them to Groq 3-step workflow:

from bdstall_chatbot_system import BDStallChatbotSystem

chatbot = BDStallChatbotSystem()

# This automatically uses Groq 3-step:
response = chatbot.process_message(
    user_id="user123",
    message="web cam lagbe",
    channel="web"
)

# This uses Bengali database (FAQ):
response = chatbot.process_message(
    user_id="user123",
    message="অর্ডার করবো কিভাবে?",
    channel="web"
)

ERROR HANDLING
════════════════════════════════════════════════════════════════════════════

Problem                    Solution
──────────────────────     ─────────────────────────────
Groq not available         → Falls back to simple keywords
API times out              → Returns "সাময়িক সমস্যা..."
No products found          → Returns "কোনো পণ্য পাওয়া যায়নি"
Response format fails       → Uses simple bullet list
Links in response          → Automatically removed
Mixed language query        → Handled gracefully
